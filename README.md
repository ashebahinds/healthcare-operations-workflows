# Healthcare Operations Workflow Project

## Overview
This project demonstrates healthcare operations analysis using a patient no-show workflow scenario.

The project includes:
- Workflow process mapping
- KPI tracking
- Operational risk analysis
- Process improvement recommendations
- Spreadsheet-based healthcare analytics

## Tools Used
- Google Sheets
- Excel
- GitHub
- Tableau

## KPIs Tracked
- No-Show Rate %
- Provider Utilization %
- Follow-Up Completion Rate %
- Reschedule Success Rate %

## Files Included
- Workflow tracker spreadsheet
- KPI dashboard
- Operational workflow documentation
- Process improvement analysis

## Objective
The goal of this project is to simulate healthcare administrative operations workflows and demonstrate analytical problem-solving skills relevant to healthcare operations and healthcare data analyst roles.
